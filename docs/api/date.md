# Date

#### Date type configs

-[**after: String**](#)
-[**before: String**](#)

---

:::caution
notice the date in `mm/dd/yyyy` format.
'after' and 'before' config just accept `String` on that format.
:::

### `after: String`
Specifies that the value must be after than `date`
### `before: String`
Specifies that the value must be before than `date`
